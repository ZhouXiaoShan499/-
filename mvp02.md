# mvp_01.md
# MindBloom 免登录本地AI情绪梳理工具 - MVP最小原型

## 一、产品定位
免登录、本地优先、零压力情绪梳理工具，专为缓解年轻人心理内耗设计。
核心：AI情绪解析 + 情绪卡片可视化网络 + 隐藏式情绪气球解压游戏，所有内容本地加密存储，实现「只对自己说话」。

## 二、MVP核心范围（必做）
1. 免登录直接使用，无注册、无授权、无云上传
2. 本地AI轻量解析：情绪标签、关键词提取
3. 可拖拽、可连线、带动画的情绪卡片网络
4. AI温和提问引导反思，无复杂操作
5. 主界面右下角「隐藏模式」按钮
6. 隐藏模式开启后主界面底部新增「解压游戏」入口
7. 情绪气球捏破解压小游戏（隐藏功能）
8. 头像右滑侧边栏：设置 + 隐藏模式总开关
9. 本地加密存储（MVP用加密JSON实现）

## 三、界面结构与交互

### 1. 主界面（AI智能体对话页）
- 顶部栏
  - 左侧：用户头像 → 点击向右滑出侧边栏
  - 中间：标题「情绪梳理」
  - 右侧：本地加密存储状态图标
- 中间区域：AI智能体对话气泡
  - 用户输入框（底部）：placeholder「随便说说吧，不用有压力…」
- 对话区下方：情绪卡片展示区
  - 自动生成情绪标签、关键词卡片
  - 卡片淡入动画，支持长按拖拽、卡片间连线建立关联
- 右下角固定按钮
  - 圆形半透明按钮：「隐藏模式」
  - 点击后：主界面底部展开「解压游戏」入口

### 2. 侧边栏（头像右滑呼出）
宽度占屏幕30%，从左向右滑入
- 设置
  - 本地存储清理
  - AI引导风格切换（温和/简洁）
  - 动画开关
- 隐藏模式
  - 总开关（与右下角按钮状态同步）
  - 说明：开启后显示解压小游戏

### 3. 隐藏模式与解压游戏
触发方式：
- 点击主界面右下角「隐藏模式」
- 或在侧边栏打开隐藏模式开关
开启后效果：
- 主界面底部新增固定 Tab：「解压游戏」
点击进入：
- 全屏半透明浮层，彩色情绪气球随机飘动
- 点击/捏破气球：破裂动画+破裂的声音
- 无计分、无排名、纯解压，可关闭返回主界面

## 四、AI本地情绪解析流程（MVP）
1. 用户输入文字
2. 本地规则引擎关键词匹配
3. 输出1~3个情绪标签 + 3~5个关键词
4. 自动生成情绪卡片并带动画
5. AI输出1条温和引导问题，不追问、不施压

## 五、本地存储规则（MVP）
- 无账号、无登录
- 本地加密JSON存储
- 存储内容：对话记录、情绪卡片与连线、设置、隐藏模式状态
- 不上传、不统计、不外泄，数据完全归属用户本地

## 六、MVP交互流程（最简路径）
1. 打开应用 → 直接进入主界面（免登录）
2. 输入情绪 → AI本地解析 → 生成情绪卡片+引导提问
3. 拖拽/连线卡片 → 构建情绪关联网络
4. 点击右下角「隐藏模式」→ 底部出现「解压游戏」
5. 进入游戏捏破气球释放压力 → 返回继续梳理
6. 点击头像右滑 → 进入设置/控制隐藏模式总开关
7. 退出应用 → 自动本地加密保存

## 七、MVP非核心（暂不实现）
- 云同步、多端同步
- 复杂AI模型、语音交互
- 社交、分享、社区
- 付费、会员、广告
- 数据报表、情绪统计图表
