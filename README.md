### 一、项目名称：情绪体验（MindBloom）

### 二、一句话slogan：可以解决年轻人心理内耗

### 三、用户痛点：
- 刷完短视频/社交媒体后感到空虚，却说不出自己真正在想什么；
- 有模糊的情绪（如疲惫、焦虑、孤独），但难以表达或梳理；
- 想记录想法，但写日记太沉重，发社交平台又像表演；
- 缺乏一个安全、无评判的空间，让思绪自由生长。

### 四、解决方案：
1. 支持语音 / 文字 3 秒快速记录模糊感受，AI 通过自然语言处理（NLP）技术自动提炼精准关键词与情绪标签；
2. AI 将用户想法转化为可拖拽、可连接的动态视觉卡片，降低表达门槛，支持用户自由构建 “思想网络”；
3. AI 以 “反思催化剂” 角色存在，通过开放式、非评判性的温和提问，激发用户深度反思，挖掘情绪背后根源，不替用户做最终判断和决策；
4. 提供灵活的记录选择权，用户可自主决定是否记录情绪感受，所有内容操作权完全归属于用户；
5. 支持用户对焦虑、恐惧等负面情绪相关想法进行单独梳理和关联，针对性化解内耗。

### 五、AI参与度：
 1. 智能解析与重构：
   - AI 深度参与用户输入的即时处理，利用 NLP 技术提炼情绪标签和关键词，基于多模态算法将抽象想法转化为具象化视觉卡片
 2. 动态连接与引导：
   - AI 分析想法间的语义关联，主动建议视觉卡片的连接路径，生成开放式提问引导用户反思；
 3. 框架与设计辅助：
   - AI 协助生成产品 MVP 设计框架、界面原型图构思、功能实现逻辑梳理；
 4. 边界界定：
   - 全程不参与情绪对错判断，不替用户生成人生建议或日记结论，仅提供技术和思维引导。

### 六、 导师招募画像
1. AI 算法 / 自然语言处理工程师：
熟悉 NLP 技术落地应用、多模态算法设计，能指导情绪文本解析、语义关联分析、AI 提问策略的优化，解决 AI 在情绪理解和引导中的技术难点；
2. 移动端 APP 开发工程师：
擅长移动端本地开发、本地数据加密存储技术，能指导免登录功能实现、视觉卡片拖拽交互的开发、本地数据安全保障等技术问题；
3. 产品交互设计师：
专注于轻量、无压力的产品交互设计，熟悉心理类 APP 的用户体验设计原则，能指导产品界面布局、交互流程优化、趣味功能（情绪气球）的设计融合；
4. 用户体验研究 / 心理学相关专业导师：
解年轻人心理内耗的核心心理机制，能指导 AI 提问的话术设计、情绪引导的节奏把控，确保产品功能符合用户心理需求，避免二次心理负担。
