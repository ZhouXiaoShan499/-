项目名称：情感共鸣体验

一句话slogan：可以解决年轻人心理内耗

用户痛点：
- 刷完短视频/社交媒体后感到空虚，却说不出自己真正在想什么；
- 有模糊的情绪（如疲惫、焦虑、孤独），但难以表达或梳理；
- 想记录想法，但写日记太沉重，发社交平台又像表演；
- 缺乏一个安全、无评判的空间，让思绪自由生长。

解决方案：
1. 3秒记录模糊感受，AI自动提炼关键词与情绪标签；
2. 将每个想法变成可拖拽、可连接、可装饰的视觉卡片；
3. AI不替你思考，而是温和提问，激发深度反思；
4. 所有想法沉淀为“思想花园”，随时间可视化你的内在成长；
5. 支持静默分享，只传递内容，不暴露自我。

AI参与度：
1. AI提供简便的答案，让用户了解自己内心的想法和感受。
2. AI婉转地提问用户，激发用户深度反思，并帮助用户梳理自己的想法和感受。
3. AI通过提问和回答的方式，帮助用户了解自己的内心世界，并找到解决问题的方法。

缺点和改进：
1. AI回答过于直接，没有给用户深度反思的机会。
2. AI提问过于宽泛，没有具体给出解决方法。
3. AI回答过于简单，没有给用户提供足够的信息和支持。

改进建议：
1. AI回答应该通过多个层级的提问和回答，逐渐给出具体的解决方法。
2. AI提问应该具体而详细，给用户提供足够的信息和支持。
3. AI回答应该通过多种方式表达，例如文字、图像、音频等，给用户提供多种选择和体验。
